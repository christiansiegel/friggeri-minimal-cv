# friggeri-minimal-cv

LaTeX CV template

<kbd><img src="example-cv.png" /></kbd>

## Usage

* Make sure *Helvetica Neue* font is installed
* Modify [example-cv.tex](example-cv.tex) to suit your needs
* Compile with XeLaTeX
